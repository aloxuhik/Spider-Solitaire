# Spider-Solitaire
Base of popular card game, created as programming exercise.
This isn't complete game, only base mechanics.

## Mechanics
- Create cards
- Shuffle and deal cards
- Drag and drop cards
- Check for finished sequence of cards
- Check if card/s can be droped on column
